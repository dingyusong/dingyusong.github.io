音视频同步

解码后的每一帧音视频都含有录制时候的时间戳，，我们需要根据时间戳将其正确的播放出来，但是在网络传输中可能会丢失一些数据，或者延时获取，这时候我们需要一定的策略来进行音视频同步。大体分为几种策略：1.缓存一定的视频数据，视频追音频等。
