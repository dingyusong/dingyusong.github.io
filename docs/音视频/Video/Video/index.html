Video

Video#

图像和屏幕的关系

  • 图像是数据
  • 屏幕是显示设备
  • 图像是经过驱动程序让屏幕显示图像

RGB色彩问题

  • RGB与BGR
  • BMP使用的是BGR格式，需要进行转换

屏幕指标

  • PPI（pixels per inch）
  • DPI（dots per inch）
  • PPI>300就属于视网膜级别

两个重要概念#

  • 分辨率
  • 帧率

分辨率

  • X轴的像素个数 x Y轴的像素个数
  • 常见的宽高比 16:9 / 4:3
  • 360P/720P/1K/2K

帧率

  • 每秒钟采集/播放图像的个数
  • 动画的帧率是25帧/s
  • 常见的帧率:15帧/s,30帧/s,60帧/s

码流#

未编码视频的RGB码流

RGB码流=分辨率（宽x高)
x 3(Byte)
x 帧率（25帧）

图像的显示

  • 图像大小等于显示区域的大小
  • 图像小于显示区域（拉伸/留白）
  • 图像大于显示区域（缩小/截断）

YUV转RGB

R = Y + 1.140*V
G = Y - 0.394*U - 0.581*V
B = Y + 2.032*U
