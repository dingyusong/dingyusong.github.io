OS X架构概述#

  • 用户体验层：包括Aqua，Dashboard，Spotlight，和辅助功能等。在iOS中，用户体验层包括SpringBoard，同时还支持Spotlight。
  • 应用架构层：包括Cocoa，Carbon，Java。（ios中只有Cocoa，严格说是Cocoa的衍生品：CocoaTouch）
  • 核心框架：有时候称为图形和媒体层。包括核心框架，OpenGL，和QuickTime。
  • Darwin：（完全开源）操作系统核心-包括内核和UnixShell环境

libSystem.B.dylibd是对libc等一系列库的封装。
