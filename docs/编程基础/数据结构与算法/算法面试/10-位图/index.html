50. 第一个只出现一次的字符位置#

题目链接#

牛客网

题目描述#

在一个字符串中找到第一个只出现一次的字符，并返回它的位置。字符串只包含 ASCII 码字符。

Input: abacc
Output: b

解题思路#

最直观的解法是使用 HashMap 对出现次数进行统计：字符做为 key，出现次数作为 value，遍历字符串每次都将 key 对应的 value 加 1。最后再遍历这个 HashMap 就可以找出出现次数为 1 的字符。

考虑到要统计的字符范围有限，也可以使用整型数组代替 HashMap。ASCII 码只有 128 个字符，因此可以使用长度为 128 的整型数组来存储每个字符出现的次数。

public int FirstNotRepeatingChar(String str) {
int[] cnts = new int[128];
for (int i = 0; i < str.length(); i++)
cnts[str.charAt(i)]++;
for (int i = 0; i < str.length(); i++)
if (cnts[str.charAt(i)] == 1)
return i;
return -1;
}

以上实现的空间复杂度还不是最优的。考虑到只需要找到只出现一次的字符，那么需要统计的次数信息只有 0,1,更大，使用两个比特位就能存储这些信息。

public int FirstNotRepeatingChar2(String str) {
BitSet bs1 = new BitSet(128);
BitSet bs2 = new BitSet(128);
for (char c : str.toCharArray()) {
if (!bs1.get(c) && !bs2.get(c))
bs1.set(c); // 0 0 -> 0 1
else if (bs1.get(c) && !bs2.get(c))
bs2.set(c); // 0 1 -> 1 1
}
for (int i = 0; i < str.length(); i++) {
char c = str.charAt(i);
if (bs1.get(c) && !bs2.get(c)) // 0 1
return i;
}
return -1;
}

我的解法#

C++#
class Solution {
public:
int FirstNotRepeatingChar(string str) {
//空字符串判断
if(str.empty()){
return -1;
}
//C++hash表怎么用
std::map<char, int> charmap;
for(int i = 0; i < str.size(); i++){
charmap[str[i]]++;
}
for(int i = 0; i < str.size(); i++){
if(charmap[str[i]] == 1)
return i;
}
return -1;
}
};

空间复杂度: O(n)

时间复杂度: O(n)

字符的个数是否是有限的。

进一步：保存对应char的次数不用std::map(时间复杂度O(log n))，而是用数组，时间复杂度O(1)

class Solution {
public:
int FirstNotRepeatingChar(string str) {
//空字符串判断
if(str.empty()){
return -1;
}
//C数组的声明和初始化，只有在初始化时，才能设为0
char chars[128] = {0};
for(int i = 0; i < str.size(); i++){
chars[str[i]]++;
}
for(int i = 0; i < str.size(); i++){
if(chars[str[i]] == 1)
return i;
}
return -1;
}
};

空间复杂度: O(1) 数组的大小是个常数

时间复杂度: O(n) 两次遍历字符串长度，字符串长度为n则是O(n)*2 = O(n)

空间复杂度优化:

使用bitset代替字符数组，空间减少1/4.

class Solution {
public:
int FirstNotRepeatingChar(string str) {
//空字符串判断
if(str.empty()){
return -1;
}
//C数组的声明和初始化，只有在初始化时，才能设为0
// char chars[128] = {0};
std::bitset<128> bs1;
std::bitset<128> bs2;
for(int i = 0; i < str.size(); i++){
if(!bs1[str[i]] && !bs2[str[i]]) {// 00 -> 10
// bs1[str[i]] = true;
bs1.set(str[i]);
}
else if(bs1[str[i]] && !bs2[str[i]]) {// 10 -> 11
// bs1[str[i]] = true;
bs2.set(str[i]);
}
}
for(int i = 0; i < str.size(); i++){
if(bs1[str[i]] && !bs2[str[i]]) {// 10 说明只有一个值
// bs1[str[i]] = true;
return i;
}
}
return -1;
}
};

空间复杂度: O(1) 数组的大小是个常数

时间复杂度: O(n) 两次遍历字符串长度，字符串长度为n则是O(n)*2 = O(n)
