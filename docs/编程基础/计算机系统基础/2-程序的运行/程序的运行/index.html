存储概念-重要#

每种语言数据类型的大小要有一个概念。1M大概能存多少int，long，char，float，double。

心中对存储的大小要有一个基本的概念：

1kB的空间 在32机器上，int4字节B，能放 1024/4 = 256。 也就是1kB能放256个int。大概有一个概念。

1M能放256个int 262144 26万级别

1M能放128个long 12万级别

1M能放1024个char。 1048576 百万级别。
