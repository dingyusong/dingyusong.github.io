toolchain#

VCPKG_CHAINLOAD_TOOLCHAIN_FILE

Specifies an alternate CMake Toolchain file to use.

This (if set) will override all other compiler detection logic. By default, a toolchain file is selected from scripts/toolchains/ appropriate to the platform.

See also the CMake documentation for toolchain files: https://cmake.org/cmake/help/v3.11/manual/cmake-toolchains.7.html.

cmake ${LOGAN_SRC_DIR} -G Xcode -DCMAKE_TOOLCHAIN_FILE="${SCRIPT_DIR}/src/LoganCore/Deps/vcpkg/scripts/buildsystems/vcpkg.cmake" -DVCPKG_CHAINLOAD_TOOLCHAIN_FILE=$SCRIPT_DIR/ios.toolchain.cmake -DPLATFORM=OS64

triplets#

如果要使用交叉编译：需要指定对应的triplet,也就是定义 VCPKG_TARGET_TRIPLET 这个变量

# 生成xcode工程
# 参考 https://github.com/leetal/ios-cmake
cmake ${LOGAN_SRC_DIR} -G Xcode \
-DCMAKE_TOOLCHAIN_FILE="${SCRIPT_DIR}/src/LoganCore/Deps/vcpkg/scripts/buildsystems/vcpkg.cmake" \
-DVCPKG_CHAINLOAD_TOOLCHAIN_FILE=$SCRIPT_DIR/ios.toolchain.cmake \
-DPLATFORM=${PLATFORM} \
-DVCPKG_TARGET_TRIPLET=${TRIPLET}

这个变量的值列表如下：

./vcpkg help triplet

相关设置信息在这里：https://vcpkg.readthedocs.io/en/latest/users/triplets/

iOS：

  • arm64-ios: arm64 架构

  • arm-ios：armv7架构

  • x64-ios：模拟器

mac：

  • arm64-osx: arm64 架构mac，m1机器

  • x64-osx: intel机器。

VCPKG_CHAINLOAD_TOOLCHAIN_FILE#

Specifies an alternate CMake Toolchain file to use.

This (if set) will override all other compiler detection logic. By default, a toolchain file is selected from scripts/toolchains/ appropriate to the platform.

See also the CMake documentation for toolchain files: https://cmake.org/cmake/help/v3.11/manual/cmake-toolchains.7.html.

交叉编译的toolchain file

#!/bin/sh
SCRIPT_DIR="$(
cd "$(dirname "$0")"
pwd
)"
echo "++++++++++++++++++++++++++开始配置环境++++++++++++++++++++++++++++++++"
echo "======================进入Deps目录==========================="
cd ${SCRIPT_DIR}/Deps
echo "======================拉取子仓库仓库=========================="
git submodule update --init --recursive
if [[ ! -d "vcpkg" ]]; then
echo "======================正在安装vcpkg=========================="
git clone https://github.com/microsoft/vcpkg
./vcpkg/bootstrap-vcpkg.sh
echo "set(VCPKG_OSX_DEPLOYMENT_TARGET 9.0)" >> ${SCRIPT_DIR}/Deps/vcpkg/triplets/community/arm64-ios.cmake
echo "set(VCPKG_OSX_DEPLOYMENT_TARGET 9.0)" >> ${SCRIPT_DIR}/Deps/vcpkg/triplets/community/arm-ios.cmake
echo "set(VCPKG_OSX_DEPLOYMENT_TARGET 9.0)" >> ${SCRIPT_DIR}/Deps/vcpkg/triplets/community/x86-ios.cmake
echo "set(VCPKG_OSX_DEPLOYMENT_TARGET 9.0)" >> ${SCRIPT_DIR}/Deps/vcpkg/triplets/community/x64-ios.cmake
fi
cd vcpkg
echo "======================安装curl openssl zlib库============================="
./vcpkg install curl:arm64-ios openssl:arm64-ios zlib:arm64-ios
./vcpkg install curl:arm-ios openssl:arm-ios zlib:arm-ios
./vcpkg install curl:x64-ios openssl:x64-ios zlib:x64-ios
./vcpkg install curl:x64-osx openssl:x64-osx zlib:x64-osx
./vcpkg install curl:arm64-osx openssl:arm64-osx zlib:arm64-osx
