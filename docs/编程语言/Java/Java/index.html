Java#

Java分三个版本：JavaSE(标准版)、JavaEE(企业版)、JavaME(微型版)，其中JavaSE是Java编程语言的基础，JavaEE是用于公司的PC端开发的，而JavaME 是用于移动端开发的。

javaee和javase的区别:https://zhuanlan.zhihu.com/p/33468599

简介#

Hello World#

public class HelloWorld
{
//java入口程序main
public static void main(String[] args)
{
System.out.println("Hello World!");
}
}
➜ HelloWorld git:(master) ✗ vi HelloWorld.java
➜ HelloWorld git:(master) ✗ javac -d . HelloWorld.java
➜ HelloWorld git:(master) ✗ java HelloWorld
Hello World!

源文件命名规则#

  1. java程序源文件后缀必须是.java（见示例2）
  2. 通常情况下java程序源文件的主文件是可以随意的。但有一种例外如果源文件里面定义了一个public类，则源文件的主文件名字必须与public类的类名相同。
  • 示例2
➜ HelloWorld git:(master) ✗ cp HelloWorld.java hello.java
➜ HelloWorld git:(master) ✗ javac -d . hello.java
hello.java:1: 错误: 类HelloWorld是公共的, 应在名为 HelloWorld.java 的文件中声明
public class HelloWorld
^
1 个错误
  • 示例3
➜ HelloWorld git:(master) ✗ cp HelloWorld.java hell.ja
➜ HelloWorld git:(master) ✗ javac -d . hell.ja
错误: 仅当显式请求注释处理时才接受类名称 'hell.ja'
1 个错误
  • 注意事项
  1. Java是严格区分大小写的语言。
  2. 一个Java源文件只定义一个类，不同的类使用不同的源文件定义。
  3. 让Java源文件名和源文件里面的public类名相同。

标识符规则#

标识符：变量，对象，类，方法命名的符号。必须以字母，下划线（_）,美元符号（$）开头，后面跟任意数目的字母，数字，下划线，美元符号。注意字母并不限于26个英文字母，中日字符都可以哦。

数据类型#

Java语言支持的类型分为两类：基本类型（primitive type）和引用类型（reference type）

  • 基本类型

    • Boolean
    • 数值类型
      • 整数类型
        • byte
        • short
        • long
        • char
      • 浮点类型
        • float
        • double 类型
    • 类
    • 接口
    • 数组类型

  • 注意：

    • 字符型 char 必须用单引号
    • String是类 是引用类型的变量所以要用"".
