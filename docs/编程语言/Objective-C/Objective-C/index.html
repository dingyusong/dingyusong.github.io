Objective-C#

[toc]

apple系官方语言，做ios，macos，watchos，tvos等应用开发必备技能，本篇总结一下OC这门语言的基础知识和特性。

Objective-C是你写OS X（maxOS）程序和iOS程序的首选语言。它作为C语言的一个超集（即C语言是它的一部分）提供了面对对象的特性和动态运行时runtime。它继承了C语言的语法，基本类型，和控制流并加入了定义类和方法的语法。他提供了动态类型和动态绑定，同时对对象图管理，object literals（极其方便地定义数组等对象的方法——一个语法糖） 提供了语言层面上的支持。

我理解的编程其实就是对数据的处理。以此发散开来，那个编程其实就是变成了对下面几个元素的处理。

  1. 数据（数据的命名，数据的生成和改变）
  2. 运算符（用来连接数据）
  3. 语句（数据，运算符，这些组成语句）
  4. 函数（数据，语句组成函数）
  5. 对象 (数据，函数组成对象)
  6. 模块

用上面的模型来理解的话，那么几乎所有的编程语言都从顶端架构去理解他。这样学习某个编程语言就变成了熟悉该语言针对向上述元素的不一样表现形式而已。

程序是指令和数据的集合。

框架不是语言，但是框架是使用语言进行编程必不可少的。

到OC这里就是，cocoa和cocoaTouch是OC框架，不是OC语言，但是是OC语言开发必不可少的。所以做开发的第一步是学习基础语法，第二步是学习基础框架。

因为一个对象所属的类在程序运行时才能被确定(Because the class of an object is determined at runtime)，所以我们在创建或使用一个实例时将变量定义成什么类型是无差别的。
