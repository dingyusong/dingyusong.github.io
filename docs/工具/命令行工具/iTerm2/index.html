Option操控#

1. “⌥+删除” 删除一个单词#

具体配置方法如下图，iTerm2 > Preferences > Profiles > Keys

在图中右下角的位置 "Left option (⌥) key act as"，勾选成 “+Esc”

即可通过 “⌥+删除” 删除一个单词

按单词后退光标#

继续在上面的图中，点击加号（+）添加一个新的 Key Mapping

点击 Keyboard Shortcut 时会录制你的操作（即此时需要同时按 ⌥ 和 ← ） Action 选择 “Send Escape Sequence” Esc+ 填入 “b” 这样就可以通过 ⌥+← 来按照单词后退光标了

3. 按单词前进光标#

和刚才的操作类似，录入 ⌥+→，并填入 “f”
