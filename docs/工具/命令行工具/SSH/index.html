服务器登录#

配置免密登录

Ubuntu上SSH免密登录:https://www.jianshu.com/p/7c86ee080e18

  • 将公钥复制到服务器
sudo vim ~/.ssh/authorized_keys
#如果没有权限则chmod 600 ~/.ssh/authorized_keys

使用vim编辑器打开.ssh目录下的authorized_keys，并且将本地的id_rsa.pub内容复制进去，如有多个公钥则换行即可，并使用wq命令保存退出。
