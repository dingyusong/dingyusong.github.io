coreutils#

GNU核心工具组

GNU核心工具组（英语：GNU Core Utilities，亦常缩写为coreutils）是一个 GNU 软件包，它包含了许多基本工具（如cat，ls 和 rm）在类 Unix 操作系统上的重新实现。

GNU 核心工具组支持以长选项作为命令调用参数。
