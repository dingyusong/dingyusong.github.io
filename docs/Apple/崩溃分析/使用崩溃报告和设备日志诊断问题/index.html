使用崩溃报告和设备日志诊断问题

使用崩溃报告和设备日志诊断问题#

要使用崩溃报告调试问题：

  1. 使用符号信息构建您的应用程序并在分发应用程序之前保留 Xcode 存档。
  2. 检索问题的崩溃报告。有关检索崩溃报告的不同方式，请参阅获取崩溃报告和诊断日志。
  3. 将十六进制地址转换为应用程序的符号名称，如将可识别的符号名称添加到崩溃报告 中所述。
  4. 确定崩溃是否符合识别常见崩溃原因中的任何模式。请参阅分析崩溃报告和检查崩溃报告中的字段以获取有关该问题的更多见解。
  5. 更新您的代码以解决问题。
  6. 使用XCTest框架添加测试以确保问题不会在未来再次发生。
