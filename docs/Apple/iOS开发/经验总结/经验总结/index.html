TargetConditionals 文件解析#

https://blog.csdn.net/u011374318/article/details/79464815

编译目标配置文件 在苹果提供的 SDK 中的 usr/include/TargetConditionals.h 文件中会自动配置编译器所要编译的代码将要使用的微处理器指令集、运行系统以及运行时环境。

TargetConditionals.h 适用于所有的编译器，但是它只能被运行于 Mac OS X 系统上的编译器所识别。

#define Weakify(o) rac_keywordify __weak typeof(o) o##Weak = o;
#define Strongify(o) rac_keywordify __strong typeof(o) o = o##Weak;
@Weakify(self)
void (^btnClick)() = ^void() {
@Strongify(self)
if (self.ningboModel.inUrl && self.ningboModel.inUrl.length > 0) {
YYIWebViewController *vc = [[YYIWebViewController alloc] init];
vc.urlString = self.ningboModel.inUrl;
vc.hidesBottomBarWhenPushed = YES;

else {

[alert show];
}
