上传#

打包完成要上传到appstore，可通过Organizer，Application Loader，或者altool，等 本文来阐述一下上传这事。

Organizer#

通过xcode的菜单栏: window->organizer->点击Distribute App->iOS AppStore 就行。

Application Loader#

使用 Application Loader 上传 App 的二进制文件

https://help.apple.com/itc/apploader/

命令行altool#

altool就是(Application loader tool)

通过 altool 上传 App 的二进制文件

altool --validate-app -f file -u username [-p password] [--output-format xml]
altool --upload-app -f file -u username [-p password] [--output-format xml]
