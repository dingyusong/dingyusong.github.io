bitcode

bitcode#

Archive 的appstore的包只有一个section

Sections64： __LLVM

➜ binary lipo RZGMS -thin arm64 -output RZGMS_arm64
➜ binary jtool2 --pages RZGMS_arm64 | grep LLVM
0x38000-0x3c000 __LLVM (16384 bytes)
0x38000-0x38001 __LLVM.__bundle (1 bytes)
➜ binary jtool2 --pages bitcode-demo-release-fastlane-appstore-enable | grep LLVM
0x62d8-0x1a2d8 __LLVM (81920 bytes)
0x62d8-0x176aa __LLVM.__bundle (70610 bytes)

使用build导出的framework

  1. LLVM的section是空的，只有一字节
➜ bitcode_demo_framework.framework jtool2 --pages bitcode_demo_framework | grep LLVM
0x10000-0x14000 __LLVM (16384 bytes)
0x10000-0x10001 __LLVM.__bundle (1 bytes)
  1. 被其他项目依赖的时候，如果其他项目开启bitcode则不能支持！

➜ binary lipo RZGMS -thin arm64 -output RZGMS_arm64
➜ binary jtool2 --pages RZGMS_arm64 | grep LLVM
0x38000-0x3c000 __LLVM (16384 bytes)
0x38000-0x38001 __LLVM.__bundle (1 bytes)
➜ binary jtool2 --pages bitcode-demo-release-fastlane-appstore-enable | grep LLVM
0x62d8-0x1a2d8 __LLVM (81920 bytes)
0x62d8-0x176aa __LLVM.__bundle (70610 bytes)
➜ binary lipo RZPaas_iOS -thin arm64 -output Paas_arm64
➜ binary jtool2 --pages Paas_arm64 | grep LLVM 0x12f4000-0x63d4000 __LLVM (84803584 bytes)
0x12f4000-0x63d0378 __LLVM.__bundle (84788088 bytes)

xcodebuild 添加 BITCODE_GENERATION_MODE=bitcode 这个编译选项，就可以了

xcodebuild BITCODE_GENERATION_MODE=bitcode -workspace "${workspace_name}" -scheme "RZGMS" ${no_debug} ONLY_ACTIVE_ARCH=NO -configuration 'Release' -sdk iphoneos BUILD_DIR="${BUILD_DIR}" clean build
# build simulator
xcodebuild BITCODE_GENERATION_MODE=bitcode -workspace "${workspace_name}" -scheme "RZGMS" ${no_debug} ONLY_ACTIVE_ARCH=NO -configuration 'Release' -sdk iphonesimulator BUILD_DIR="${BUILD_DIR}" clean build
