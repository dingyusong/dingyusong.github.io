Reveal

Reveal#

通过Cydia 安装 Reveal2Loader

Dby6S:~ root# find / -name "*Reveal*"
/System/Library/PrivateFrameworks/RevealCore.framework
/Library/PreferenceLoader/Preferences/Reveal2Loader.plist
/Library/Frameworks/RevealServer.framework
/Library/Frameworks/RevealServer.framework/RevealServer
/Library/Frameworks/RevealServer.framework/Headers/RevealServer.h
/private/var/mnt/System/Library/PrivateFrameworks/RevealCore.framework
/private/var/mobile/Containers/Data/Application/97CFFB66-37AE-4234-8A67-8F994B1FAF56/Library/Caches/MappedImageCache/com.apple.TelephonyUI.TPRevealingRingView
/private/var/mobile/Containers/Data/Application/D90E5BD3-7C31-4092-8A32-E1B7EC7B18A1/Library/Caches/Reveal
/private/var/mobile/Library/Caches/MappedImageCache/com.apple.TelephonyUI.TPRevealingRingView
/private/var/mobile/Library/Preferences/com.rheard.RHRevealLoader.plist

需要替换RevealServer这个动态库，或者替换整个RevealServer.framework，RevealServer.framework 从mac上拿到。

然后重启

Dby6S:~ root# killall SpringBoard

Reveal 调试越狱机第三方应用 UI

破解版#

https://macwk.com/soft/reveal

通过 Sileo 下载 CPRevealLoader 然后替换掉就行了。
